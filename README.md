📊 A hands-on repository for learning Pandas DataFrame operations in Python.
This project demonstrates how to create, manipulate, clean, and analyze structured datasets using one of the most powerful Python data analysis libraries.

📊 Example DataFrame
import pandas as pd

data = {
    "Name": ["Alice", "Bob", "Charlie"],
    "Age": [24, 30, 22],
    "City": ["New York", "London", "Tokyo"]
}

df = pd.DataFrame(data)

print(df)

Output:

      Name  Age      City
0    Alice   24  New York
1      Bob   30    London
2  Charlie   22     Tokyo
